# UI/UX Design Project

## Project Title
Fitness Mobile App UI Design

## Overview
This project is a modern and user-friendly mobile application UI designed using Figma. The main goal is to create a clean, attractive, and easy-to-use fitness application that helps users achieve their health and fitness goals.

## Features
- Modern and minimal UI
- User-friendly navigation
- Login & Sign Up screens
- Home Dashboard
- Workout Categories
- Workout Details
- Progress Tracking
- Profile Screen
- Responsive mobile design

## Tools Used
- Figma
- Material Design Icons
- Google Fonts

## Design Process
1. User Research
2. Wireframing
3. High-Fidelity UI Design
4. Prototyping
5. Final Design
